# Instant Landing Page

Describe your idea. Get a live landing page in 60 seconds.

## Run locally

```
cd instant-landing-page
npm install
npm start
```

Open http://localhost:3000. Type an idea, click Generate, see the page render in the preview panel.

Set `ANTHROPIC_API_KEY` (see `.env.example`) to generate pages with Claude. Without it, a built-in template generator produces a working page so the app runs with zero setup.
